# Employee Attrition Analysis for HR

This project provides a Streamlit web app for HR analytics with:
- employee turnover KPIs
- attrition trends by department and salary band
- factors influencing resignation such as age, salary, and tenure
- a classification model to predict attrition

## Run locally

1. Install dependencies:
   pip install -r requirements.txt
2. Start the app:
   streamlit run app.py

## Verification

Run the tests with:

pytest -q

The app uses a synthetic HR analytics sample designed for demonstration and modeling practice.
